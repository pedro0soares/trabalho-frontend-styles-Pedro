Entrega rápida do projeto organizada por Gepeto.

Estrutura:
- frontend/   (Vite + React)
- backend/    (Express + SQLite para testes)

Como rodar:
1) Backend:
   cd backend
   npm install
   npm start
   Usuário demo: demo@demo.com / senha

2) Frontend:
   cd frontend
   npm install
   npm run dev
   Abra http://localhost:5173 (vite padrão)

O frontend usa http://localhost:4000 como API.

Obs: Ajuste conforme necessário. Boa sorte!
